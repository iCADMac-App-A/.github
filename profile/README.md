## Overview

<a href="#">iCADMac</a> provides a professional CAD environment for Mac with full DWG file compatibility and an AutoCAD-compatible command set. Engineers, architects, and technical designers who work on Mac and need to exchange files with AutoCAD users find iCADMac delivers the compatibility and capability that professional CAD work requires without the Windows dependency that AutoCAD itself has traditionally demanded.

The <a href="#">DWG format support</a> reads and writes the native AutoCAD file format with high fidelity. Files created in iCADMac open correctly in AutoCAD and vice versa, maintaining geometry, layers, blocks, dimensions, and other drawing data through the exchange. <a href="#">DXF compatibility</a> covers older exchange formats for workflow compatibility with legacy systems and other CAD applications that use DXF for interoperability.

<a href="#">AutoCAD-compatible commands</a> allow users familiar with AutoCAD to work productively immediately without learning a new command vocabulary. Type familiar command names at the command line and iCADMac executes the expected operations. This command compatibility extends to command aliases, command line history, and the general interaction model that AutoCAD users have developed through years of practice. <a href="#">Lisp scripting support</a> runs AutoLISP routines developed for AutoCAD, enabling use of existing automation scripts.

The <a href="#">2D drafting tools</a> cover the complete set of operations technical drawing requires — line, arc, circle, polyline, spline, hatch, dimension, text, and leader tools all operate with the precision that engineering documentation demands. <a href="#">Constraint-based drafting</a> applies geometric and dimensional constraints to maintain design intent through modifications. These constraints ensure that changing one dimension updates related geometry correctly.

## Key Features

- <a href="#">Full DWG compatibility</a> reads and writes native AutoCAD format preserving all drawing data accurately
- <a href="#">AutoCAD command set</a> accepts familiar command names for immediate productivity without relearning
- <a href="#">AutoLISP scripting</a> runs existing AutoCAD automation routines without modification on Mac
- <a href="#">Complete 2D drafting tools</a> including line arc circle polyline spline hatch dimension and text
- <a href="#">Geometric constraints</a> maintain design intent by linking related geometry through parametric rules
- <a href="#">3D solid modeling</a> creates three-dimensional objects through extrusion boolean and profile operations
- <a href="#">Layer management system</a> organizes drawing elements with color visibility and state configuration
- <a href="#">Block library tools</a> store and insert reusable drawing components across projects consistently
- <a href="#">Professional plot output</a> configures scaled printed drawings with correct dimension and line weight rendering
